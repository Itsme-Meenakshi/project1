# 🐶🐱 Cat and Dog Detection using CNN

A Deep Learning project that classifies images as either **Cat** or **Dog** using a Convolutional Neural Network (CNN).

This project is implemented using **Python, TensorFlow/Keras, and OpenCV** inside Jupyter Notebook.

---

## 📌 Project Overview

The model is trained on image datasets of cats and dogs to learn visual features and perform binary image classification.

The project demonstrates:

- Image preprocessing
- CNN model building
- Deep learning-based classification
- Model training and testing
- Prediction on custom images

---

## 🚀 Technologies Used

## 🧠 Model Architecture

The CNN model consists of:

- Convolution Layers
- Max Pooling Layers
- Flatten Layer
- Dense Layers
- Dropout Layer
- Sigmoid Activation Function
